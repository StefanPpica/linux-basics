| Comandă | Descriere | Exemplu |
| :--- | :--- | :--- |
| `ls` | Listează fișierele | `ls -la` |
| `cd` | Schimbă directorul | `cd /var/log` |
| `mkdir` | Creează un folder | `mkdir proiect-nou` |
| `touch` | Creează un fișier gol | `touch lectie.md` |
| `cat` | Afișează conținutul unui fișier | `cat lectie.md` |
| `cp` | Copiază un fișier (sursă -> destinație) | `cp lectie.md /home/stef/Documents/Lectii` |
| `mv` | Redenumește sau mută fișiere/foldere | `mv oldfile newfile` |
| `rm` | Șterge un fișier | `rm test.txt` |
| `find` | Căutare după nume și tip | `find /home -type d -name MyFolder` |
| `help` | Ajutor comenzi | `help echo` sau `ls --help` |
| `man` | Manual instrucțiuni | `man ls` |
| `whatis` | Descriere scurtă a comenzii | `whatis cat` |
| `alias` | Redenumire/Scurtătură comenzi | `alias ll='ls -la'` |
