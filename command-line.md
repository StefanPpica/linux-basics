# Comenzi de Bază în Linia de Comandă

| Comandă | Nume | Descriere | Exemplu |
| :--- | :--- | :--- | :--- |
| `ls` | **List** | Listează fișierele și directoarele din locația curentă. | `ls -la` |
| `cd` | **Change Directory** | Schimbă folderul curent către o altă cale. | `cd /var/log` |
| `mkdir` | **Make Directory** | Creează un folder (director) nou. | `mkdir proiect-nou` |
| `touch` | **Touch** | Creează un fișier nou gol sau actualizează data accesării. | `touch lectie.md` |
| `cat` | **Concatenate** | Afișează conținutul întreg al unui fișier în terminal. | `cat lectie.md` |
| `cp` | **Copy** | Copiază fișiere sau directoare de la sursă la destinație. | `cp file.txt /backup/` |
| `mv` | **Move / Rename** | Mută un fișier sau îl redenumește dacă destinația e în același folder. | `mv vechi.txt nou.txt` |
| `rm` | **Remove** | Șterge un fișier (atenție: ștergerea este definitivă). | `rm test.txt` |
| `find` | **Find** | Caută fișiere sau foldere după nume, tip sau dimensiune. | `find . -name "*.md"` |
| `man` | **Manual** | Deschide manualul de utilizare al unei comenzi. | `man ls` |
| `help` | **Help** | Afișează un ajutor rapid pentru comenzile interne. | `ls --help` |
| `alias` | **Alias** | Creează o scurtătură (poreclă) pentru o comandă lungă. | `alias ll='ls -la'` |
